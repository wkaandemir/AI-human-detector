# Güvenlik Politikası

## Desteklenen Sürümler

Projenin hangi sürümlerinin güvenlik güncellemeleriyle desteklendiğini görmek için aşağıdaki tabloyu inceleyebilirsiniz.

| Sürüm   | Destekleniyor mu?  |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Güvenlik Açığı Bildirme

Yazılımımızın güvenliğini ciddiye alıyoruz. Eğer bir güvenlik açığı bulduğunuzdan şüpheleniyorsanız, lütfen bunu hemen bize bildirin.

### Nasıl Bildirilir?

Lütfen güvenlik açıkları için GitHub üzerinde herkese açık bir `Issue` (Sorun) **oluşturmayın**. Bunun yerine, lütfen proje sahiplerine doğrudan e-posta gönderin.

Bildiriminizi 48 saat içinde onaylayacak ve çözüm için tahmini bir süre bildireceğiz.
